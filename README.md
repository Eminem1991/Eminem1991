## 🚀 Главный проект в моем портфолио

### 🏋️‍♂️ [HomeFit — Интеллектуальная Фитнес-Платформа (SPA)](https://github.com/Eminem1991/EminAliyev)
Полноценная фитнес-экосистема «3 в 1» (Рацион + Тренинг + Наставник), написанная на **Vanilla JS** без использования тяжелых фреймворков. 

* **Умная адаптация:** Движок квиза с мультивыбором на лету перестраивает базу данных диет под предпочтения пользователя, дефицит воды и аллергены.
* **Безопасный биллинг:** Клиентская валидация карт по **математическому алгоритму Луна** и симуляция банковского окна 3D Secure (SMS-код).
* **SMTP-верификация:** Генерация одноразовых кодов безопасности и отправка реальных писем через **EmailJS API** прямо с фронтенда.
* **Чат-центр:** Встроенный интерактивный мессенджер куратора и генерация глубоких ссылок (**Deep Linking**) для мгновенного перехода в Telegram и WhatsApp боты.
